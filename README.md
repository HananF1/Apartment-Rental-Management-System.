 🧠 LifeAI — AI Life Manager
A modern **AI-powered Life Management Web Application** designed to help users organize their daily life, track habits, monitor health, manage goals, and control smart devices — all in one place.

 🚀 Features
 📊 Dashboard
* Personalized greeting system
* AI-generated daily insights
* Overview of tasks, habits, and goals
* Weekly activity tracking

 📅 Schedule Management
* Weekly calendar view
* Add and manage events
* Timeline visualization of daily activities

 🔁 Habit Tracker
* Track daily habits (Mon–Sun)
* Automatic streak calculation 🔥
* Weekly completion rate
* Visual progress indicators

 ❤️ Health Monitoring
* Steps tracking with progress bar
* Water intake tracking 💧
* Heart rate & sleep monitoring
* Interactive health controls

 🎯 Goal Management
* Create long-term goals
* Add milestones for each goal
* Track progress dynamically
* Categorize goals

 🤖 AI Assistant
* Smart suggestions based on user behavior
* Daily productivity insights

 🏠 Smart Devices
* Control virtual smart devices:
  * 💡 Light (brightness control)
  * 🌡️ Thermostat (temperature control)
  * 🔊 Speaker (volume control)

 🧩 Technologies Used
* **HTML5** — Structure
* **CSS3** — Styling & UI Design
* **JavaScript (ES6+)** — Logic & OOP Architecture

 🏗️ Object-Oriented Design
This project follows strong **OOP principles**:

 🔒 Encapsulation
* Private fields (`#`) used to protect data
* Controlled access via getters/setters

 🧬 Inheritance
* Class hierarchy:
```plaintext
Observable → BaseModel → TrackedItem → Habit / Goal
```
 🔄 Polymorphism
* Methods like:
  * `getProgress()`
  * `getStatus()`
* Implemented differently across classes

 🧩 Abstraction
* Abstract classes:
  * `TrackedItem`
  * `SmartDevice`
* Enforce implementation in subclasses
